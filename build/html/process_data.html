process_data module¶

class process_data.ProcessDataPd(file_path: str)¶

Bases: object

A class to load and process data from a CSV file. This class provides methods to load data, discard columns by index, filter columns by variance, create the upper triangle of the correlation matrix, and get sorted correlations from the correlation matrix.

discard_columns_by_index(indices: list)¶

Discard columns by their indices. e.g. remove columns with constant values [0, 1, 2] will remove the first three columns.

euclidean_distance(col1: str, col2: str)¶

Calculate the Euclidean distance between two columns.

filter_by_variance(threshold: float)¶

Filter columns by variance. e.g. remove columns with variance below the threshold.

get_sorted_correlations(save_to_txt=False)¶

Get sorted correlations from the correlation matrix.

Parameters: correlation_matrix (DataFrame): The correlation matrix.

Returns: Series: Sorted correlations with absolute values.

load_data(index_col=None)¶

Load data from a CSV file with an optional index column.

upper_correlation_triangle()¶

Create the upper triangle of the correlation matrix.
